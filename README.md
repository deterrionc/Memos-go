Memos 是一款开源的 [flomo](https://flomoapp.com/) 替代工具，为了快速方便的部署属于自己的碎片化知识管理工具。

## 🎯 产品意图

- 📅 用于记录：每日/周计划、💡 突发奇想、📕 读后感...
- 🏗️ 代替了微信“文件传输助手”；
- 📒 打造一个属于自己的轻量化“卡片”笔记簿；

## ✨ 特色亮点

- 🦄 开源项目；
- 😋 精美且细节的视觉样式；
- 📑 体验优良的交互逻辑；
- ⚡️ 快速地私有化部署；
